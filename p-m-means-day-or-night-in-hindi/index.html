In India, there are two types of time systems – the 12-hour clock and the 24-hour clock. The 12-hour clock system is commonly used in everyday life, while the 24-hour clock system is used in official documents and other formal situations. When it comes to the 12-hour clock system, the term p.m. is used to refer to the time period between noon and midnight. In this article, we will discuss what p.m. means day or night in Hindi and how to use it correctly.

What Does p.m. Mean?

The term p.m. stands for post meridiem, which is Latin for "after midday." In the 12-hour clock system, p.m. is used to refer to the time period between noon and midnight. It is the opposite of a.m., which stands for ante meridiem and is used to refer to the time period between midnight and noon.

How to Use p.m. in Hindi

If you want to use p.m. in Hindi, you can say दोपहर से रात तक (dopahar se raat tak). This literally translates to "from noon until night" and is the equivalent of p.m. in Hindi.

Step-by-Step Guide to Using p.m. in Hindi

Here are the steps to follow when using p.m. in Hindi:

  1. Start by stating the time in Hindi. For example, if it is 3:00 p.m., you can say तीन बजे (teen baje).
  2. Next, add the phrase दोपहर से रात तक (dopahar se raat tak) to indicate that it is p.m. For example, you can say तीन बजे दोपहर से रात तक (teen baje dopahar se raat tak).
  3. Finally, you can add any additional information about the time, such as the date or the day of the week.

Tips for Using p.m. in Hindi

Here are some tips to keep in mind when using p.m. in Hindi:

  • Always use the 12-hour clock system when using p.m. in Hindi.
  • Make sure you use the correct word order when using the phrase दोपहर से रात तक (dopahar se raat tak).
  • Be sure to specify the time before using the phrase दोपहर से रात तक (dopahar se raat tak).

FAQs

What is the difference between a.m. and p.m.?

The term a.m. stands for ante meridiem and is used to refer to the time period between midnight and noon. The term p.m. stands for post meridiem and is used to refer to the time period between noon and midnight.

Yes, you can use p.m. in formal situations when referring to the time period between noon and midnight.

No, you do not need to use p.m. in Hindi. However, it can be helpful to know how to use it in case you need to communicate with someone who is more familiar with the 12-hour clock system.
